# Stage 11C expansion note

Batch C should not only handle "monsters drop loot."

It should support:
- biological harvests
- mineral and environmental extractions
- anomalous and metaphysical resources
- site-specific treasures
- relic-bearing infrastructures
- rare object-bond mechanics across weapons, armor, tools, instruments, and hybrid gear

The governing design rule is:

**specific without being brittle**

That means the packet needs categories and procedures broad enough to absorb future sourcebooks, but concrete enough that the system still feels real.
