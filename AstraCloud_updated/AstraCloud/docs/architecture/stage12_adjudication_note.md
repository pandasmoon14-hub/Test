# Stage 12 adjudication note

The purpose of this pass is not to declare RHBF "right."
It is to choose the Astra-facing terms that remain mechanically useful after the donor has been stripped of source-specific cosmology.

## Chosen doctrine

### Presence Pressure
Chosen over Aura because it preserves the function of a visible/intangible power-weight metric without binding Astra to a generic fantasy aura term.

### Fate
Chosen over Luck because it supports a larger destiny-facing subsystem and fits Astra's tone better.

### Meridian Reserve
Chosen as the generalized reserve term because it can comfortably host mystical, cultivation, techno-metaphysical, and hybrid energy expenditures without sounding locked to RHBF's source cosmology.

### Meridian Channels
Chosen as the first-pass explanation of the body-slot Technique architecture because it aligns with Astra's internal metaphysical style while preserving the donor's slot logic.

### Stronghold Phase / Technique Repository / Gate Array
Promoted because they are structurally useful and not overly source-bound.

## Explicit rejection

Astra does not inherit RHBF's full cultivation-versus-technology absolutism.
That ideological conflict may remain as a setting option or factional posture, but it is blocked from automatic canon status.
