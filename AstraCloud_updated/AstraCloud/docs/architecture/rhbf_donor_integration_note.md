# RHBF donor integration architecture note

The merge strategy is intentionally asymmetric.

RHBF is currently the strongest donor for:
- cultivation-compatible player chassis
- domain-gated technique systems
- encounter-to-harvest-to-upgrade loops
- world activity structure with hubs, trials, and repeatable realms

RHBF is not automatically authoritative for:
- final Astra cosmology
- final Astra anti-technology doctrine
- every final proper noun
- every metaphysical naming layer

Therefore the merge model is:

1. preserve strong mechanical skeletons
2. rewrite source-facing names into Astra-compatible terms
3. quarantine unstable naming and ideology in adjudication files
4. promote only the stable and reusable outputs into canonical lexicon and mechanics files
