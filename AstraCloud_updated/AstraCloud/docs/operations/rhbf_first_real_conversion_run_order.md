# RHBF first real conversion run order

## Goal

Run RHBF as the first serious donor stream in a sequence that stabilizes the Astra framework instead of flooding it.

## Pass order

### Pass 1 — Chassis
Convert:
- character creation
- progression
- attributes
- resistances
- paths
- techniques
- damage and crit rules

Outputs:
- rulebook packet updates
- mechanics map updates
- lexicon deltas
- canon candidate packets

### Pass 2 — World activity
Convert:
- hub logic
- Transit Seal gating
- Ascension Trials
- Hunting Realms
- faction pressure
- Stronghold Phase

Outputs:
- setting and activity loop packet updates
- world-side lexicon deltas
- canon conflicts for ideology/proper noun carryover

### Pass 3 — Creatures and Bound Relics
Convert:
- creature taxonomy
- statblocks
- harvest loops
- natural treasures
- Bound Relic progression

Outputs:
- creature mechanics packet
- relic mechanics packet
- harvest loop packet
- taxonomy and naming adjudications

## Promotion rule

No unstable term should move from provisional to canonical without review.
