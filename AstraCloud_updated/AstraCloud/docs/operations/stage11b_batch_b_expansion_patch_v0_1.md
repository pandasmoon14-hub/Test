# Stage 11B — Batch B Expansion Patch v0.1

This patch thickens the live Astra activity-engine packet where it was distinct but underexplained.

## What changed

The packet now explains:
- what characters actually do at a Nexus Hub
- how Transit Seals are gained and spent
- how to build an Ascension Trial
- how Hunting Realms differ from Trials
- what factions practically do to the loop
- what actions occur during Stronghold Phase

## Why this patch exists

The original packet had the right bones, but not enough procedure.
This patch makes the system easier to run at the table and easier to convert consistently later.
