# Stage 11C — Batch C Expansion Patch v0.1

This patch expands the creature / harvest / relic packet in the places most likely to matter across a large multi-system donor stream.

## What was kept stable

- creature stat block template
- Ascendant Beast concept
- harvest loop as encounter-to-progression bridge

## What was expanded

- Natural Resources
- Bound Relics
- Relic Traits

## Why this patch exists

The original Batch C packet had the right chassis, but these sections needed more extensibility to survive later donor influx from systems with:
- exotic resources
- weird materials
- unusual relic classes
- setting-specific weapons, armor, tools, or instruments
- stranger upgrade economies
