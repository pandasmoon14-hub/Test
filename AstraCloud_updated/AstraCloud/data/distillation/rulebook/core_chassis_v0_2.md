## Cross-donor chassis stance

Astra supports **stacked build layers** rather than one universal donor package chain.

A lawful player build may include several distinct layers, depending on donor pressure and campaign frame:
- body / origin layer,
- culture / background layer,
- role / path / profession layer,
- access or special-channel layer,
- and capability / asset layer.

This means Astra does **not** assume that every donor expresses player identity through the same single-chain structure. Some donors heavily weight Path-style packages. Others heavily weight licenses, contacts, resources, access channels, augmentations, bonded assets, or comparable build pressures. Astra must host all of these without flattening them into one monoculture.

## Chassis summary guidance

### Core identity
An Ascendant is a player-grade actor whose actions can materially redirect danger, opportunity, and world pressure.

Astra does not assume a single universal origin story for all Ascendants. Characters may emerge through cultivation, institutional training, catastrophe, augmentation, bonded inheritance, survival pressure, social selection, faction sponsorship, or other lawful donor-local routes.

### Current live profile
Astra's current live player profile remains D20-forward, but that is a **live implementation profile**, not proof that only one resolution architecture is lawful in Astra.

### Access law
Astra treats access as a real build and rules layer. A character may be shaped by domains, licenses, interface states, bonded permissions, augment channels, credentials, lineage privileges, or equivalent donor-local access structures.

Capability is not identical to permission. Training is not identical to access.

### Ability / technique capacity architecture
Astra owns the **grammar** of capacity, access, readiness, and activation.

Donors may realize that grammar through different lawful capacity profiles, including:
- fixed slot structures,
- prepared or readied limits,
- learned-known limits,
- rating-gated access,
- installable or bonded capacity tracks,
- or other donor-local architectures.

Exact body-slot topologies, exact fixed-capacity ladders, and other highly donor-shaped patterns are retained as donor-local profile examples rather than Astra default law.

## Starter Path exemplars

These are **starter exemplars**, not an exhaustive or universal baseline roster.
They remain useful because they show how Astra can present role-facing chassis packages, but they do not imply that all lawful Astra builds reduce to these four path patterns alone.

### 1. Astra Warpath
A frontline martial Path derived from early armament-centered donor pressure.

**Role:** battlefield pressure, mobility, positional control, precision offense.

**Starting Domains:** Armament (Martial), Armament (Simple), Armament (Ranged), Armor (Light), Armor (Medium), Space, Body.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 10, Flow 16, Core 14, Insight 10, Resonance 12, Intent 10.

**Additional Attribute Points:** +2 Flow, +1 Vessel.

**Starting Techniques / abilities:** Aim, Tactical Awareness, Portal-Step, Quick Fire.

**Astra reading:** a fast, disciplined combatant who weaponizes motion, angles, and field awareness.

### 2. Meridian Warden
A healer-support Path derived from restoration and sustain pressure.

**Role:** restoration, sustain, bodily repair, team endurance.

**Starting Domains:** Armament (Simple), Armor (Light), Magic, Life, Body.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 8, Flow 10, Core 12, Insight 14, Resonance 16, Intent 12.

**Additional Attribute Points:** +1 Insight, +2 Resonance.

**Starting Techniques / abilities:** Agare's Mark, Regrow Limb, Vital Renewal, Tranquil Embrace.

**Astra reading:** a life-channeling path focused on restoration, support, and controlled energy flow.

### 3. Rotweaver
A decay / poison / curse Path derived from attrition and condition pressure.

**Role:** attrition, conditions, battlefield weakening, toxic denial.

**Starting Domains:** Armament (Simple), Armor (Light), Death, Body, Poison, Corrosion.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 8, Flow 10, Core 12, Insight 14, Resonance 16, Intent 12.

**Additional Attribute Points:** +1 Insight, +2 Resonance.

**Starting Techniques / abilities:** Ulcerate, Thagg's Weakness, Melt Flesh, Cleanse.

**Astra reading:** a slow-kill controller using venom, corrosion, curses, and condition pressure.

### 4. Void Savant
A war-savant / arcane-artillery Path derived from elemental and high-channel pressure.

**Role:** long-range force projection, elemental shaping, battle sorcery, flexible utility.

**Starting Domains:** Magic plus one elemental domain, with Body or Mind as secondary support.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 8, Flow 10, Core 12, Insight 14, Resonance 16, Intent 12.

**Additional Attribute Points:** +1 Insight, +2 Resonance.

**Starting Techniques / abilities:** elemental bolt equivalent, battlefield utility equivalent, defensive shaping technique, one major finisher.

**Astra reading:** a disciplined reality-shaper using codified workings and high-channel expression.

## Explicit scope cautions

- These starter paths are exemplars, not a claim that all Astra chassis are path-primary in the same way.
- Donor-local capacity profiles such as exact body-slot topology or exact fixed technique ladders are not promoted here as universal Astra law.
- Social-capital pressure, resource pressure, access-channel pressure, augmentation pressure, and bonded-asset pressure remain lawful build influences even when this file does not model them in full.
