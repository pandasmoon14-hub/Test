# RHBF Batch C — Astra Creature and Relic Rules Packet v0.1

## Creature families

Astra uses creature families as active mechanics, not decorative taxonomy.
Each family may imply one or more default interactions with damage, conditions,
harvest outputs, and relic trait crafting.

Starter family set:
- Humanoid
- Beast
- Aberrant Entity
- Forged Entity
- Living Flora Entity
- Elemental Entity
- Revenant Entity
- Astral Entity

## Astra creature frame

Use the following field order when converting donor stat blocks:
- Name
- Family / secondary family
- Attribute line
- Number
- Presence Pressure
- Core / Flow / Resonance Resistances
- Vitality
- Defense Threshold
- Speed
- Stealth
- Size
- Actions
- Traits
- Attacks / abilities
- Trophy / harvest / special reward hooks

## Ascendant Beasts

Some beasts may ascend beyond instinct and develop humanlike or otherwise
sapient forms while retaining beast-origin traits and transformation capacity.
This state is treated as an Ascendant Beast breakthrough, not a separate species.

## Harvest loop

Harvest remains typed by creature family.
The Astra framework should preserve these distinctions because they drive:
- alchemy
- cooking
- relic feeding
- trade
- trophy systems

## Bound Relics

Bound Relics are owner-aligned weapons or implements that grow through
Resonant Feeding. Their traits are not random enchantments; they are crafted,
fed, and awakened through typed materials, natural treasures, and specific
component families.
