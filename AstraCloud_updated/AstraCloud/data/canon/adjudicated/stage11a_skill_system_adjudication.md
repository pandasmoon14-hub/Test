# Stage 11A skill system adjudication

## Decision

Astra formally includes:
- Attribute Checks
- Skill Checks
- Opposed Checks

Skill Checks use:

**D20 + governing Attribute modifier + Skill Rank bonus + situational modifiers**

## Skill ranks

- Untrained: +0
- Trained: +2
- Expert: +4
- Ascendant: +6

## Important doctrinal distinction

Skills are persistent trained competencies.
Techniques are activated supernatural or combat-capable expressions.

That distinction should remain stable in future Astra-facing packets.
