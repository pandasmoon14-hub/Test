# First Production Run Checklist

## Before run
- Confirm Stages 1–12 are installed.
- Confirm Stage 12 promoted terms are in place.
- Confirm blocked terms are understood.
- Confirm donor PDFs or normalized inputs are present.
- Confirm run profile and manifest template are copied into the real AstraCloud repo.

## Batch A
- Run chassis conversion first.
- Review Path, Technique, Presence Pressure, Fate, and Meridian Reserve usage.
- Confirm point-buy, Resistance, and crit rules were preserved.

## Batch B
- Run activity engine conversion second.
- Review Nexus Hub, Transit Seal, Ascension Trial, Hunting Realm, and Stronghold Phase usage.
- Confirm donor proper nouns were not silently canonized.

## Batch C
- Run creatures and relics conversion third.
- Review Ascendant Beast, Bound Relic, Relic Trait, and harvest requirement specificity.
- Confirm stat-block order and typed-damage posture were preserved.

## After run
- Review validator reports.
- Review lexicon delta packets.
- Review canon candidate packets.
- Promote only stable terms.
