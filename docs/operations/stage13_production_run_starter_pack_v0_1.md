# Stage 13 — Production Run Starter Pack v0.1

This stage turns the earlier doctrine, donor packs, and adjudication layers into a practical first-run operating bundle.

## Objective

Support the first serious donor-to-Astra production pass with:
- clear batch order
- explicit run profile expectations
- structured manifest inputs
- standardized output/report expectations
- checkpoint and review gates

## Intended use

Use this pack after Stages 1–12 are installed and verified.
It assumes the repo already contains the RHBF donor profiles, merged guidance, validator targets, and adjudicated term promotions.

## Production pass philosophy

Do not flood the pipeline.
Run in stable, reviewable bands:

1. Chassis and player core
2. World activity and faction loops
3. Creatures, harvests, and Bound Relics
4. Lexicon promotion and canon adjudication review
