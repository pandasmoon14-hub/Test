# Post-Run Review Walkthrough

## Immediate review order

1. validator concerns
2. blocked-term leakage
3. lexicon delta candidates
4. canon conflicts
5. packet readability as standalone Astra text

## Promotion rule

Do not promote a term just because it sounds cool.
Promote it only if it is:
- mechanically useful
- repeated consistently
- not overly donor-bound
- not in conflict with already promoted doctrine

## If a batch is messy

Do one of three things:
- rerun the same batch with stricter terminology discipline
- leave the term provisional and continue
- halt for adjudication if the problem touches core doctrine
