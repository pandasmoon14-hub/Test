# Stage 11 — Live Conversion Seed Pack v0.1

This is the first pack in the sequence that is primarily **converted Astra-facing text** instead of framework scaffolding.

## Scope

This pack turns the merged RHBF donor doctrine into four live conversion packets:

1. Astra player core rules
2. Astra Paths and Techniques
3. Astra activity engine
4. Astra creatures and Bound Relics

## Goal

The goal is not to reproduce RHBF phrasing. The goal is to prove that the donor skeleton can be rewritten into coherent Astra-facing text while preserving:
- D20 chassis clarity
- six-attribute structure
- three Resistances
- Path/Technique identity
- world activity loops
- encounter/harvest/relic progression
