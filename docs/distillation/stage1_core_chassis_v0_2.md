## Four starter Paths

### 1. Astra Warpath
A frontline martial Path derived from the martial/ranged chassis represented by Sharpshooter and other armament-centered classes.

**Role:** battlefield pressure, mobility, positional control, precision offense.

**Starting Domains:** Armament (Martial), Armament (Simple), Armament (Ranged), Armor (Light), Armor (Medium), Space, Body.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 10, Flow 16, Core 14, Insight 10, Resonance 12, Intent 10.

**Additional Attribute Points:** +2 Flow, +1 Vessel.

**Starting Techniques:** Aim, Tactical Awareness, Portal-Step, Quick Fire.

**Astra reading:** a fast, disciplined combatant who weaponizes motion, angles, and field awareness.

### 2. Meridian Warden
A healer-support Path derived from Lifewarden.

**Role:** restoration, sustain, bodily repair, team endurance.

**Starting Domains:** Armament (Simple), Armor (Light), Magic, Life, Body.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 8, Flow 10, Core 12, Insight 14, Resonance 16, Intent 12.

**Additional Attribute Points:** +1 Insight, +2 Resonance.

**Starting Techniques:** Agare's Mark, Regrow Limb, Vital Renewal, Tranquil Embrace.

**Astra reading:** a life-channeling path focused on restoration, support, and controlled essence flow.

### 3. Rotweaver
A decay / poison / curse Path derived from Plague Shaman.

**Role:** attrition, conditions, battlefield weakening, toxic denial.

**Starting Domains:** Armament (Simple), Armor (Light), Death, Body, Poison, Corrosion.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 8, Flow 10, Core 12, Insight 14, Resonance 16, Intent 12.

**Additional Attribute Points:** +1 Insight, +2 Resonance.

**Starting Techniques:** Ulcerate, Thagg's Weakness, Melt Flesh, Cleanse.

**Astra reading:** a slow-kill controller using venom, corrosion, curses, and condition pressure.

### 4. Void Savant
A war-savant / arcane artillery Path derived from Ekosian Thaumaturge plus the donor book’s high-magic technique structure.

**Role:** long-range mystical offense, elemental shaping, battle sorcery, flexible utility.

**Starting Domains:** Magic plus one elemental domain, with Body or Mind as secondary support.

**Starting Resistances:** +1 Flow Resistance, +1 Resonance Resistance.

**Recommended Attributes:** Vessel 8, Flow 10, Core 12, Insight 14, Resonance 16, Intent 12.

**Additional Attribute Points:** +1 Insight, +2 Resonance.

**Starting Techniques:** elemental bolt equivalent, battlefield utility equivalent, defensive shaping technique, one major finisher.

**Astra reading:** a disciplined reality-shaper using codified workings and high Resonance checks.
