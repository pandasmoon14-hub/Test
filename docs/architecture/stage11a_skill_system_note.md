# Stage 11A skill system note

The Astra skill system should not fight the donor chassis.
RHBF already leans heavily on plain attribute checks plus situational skill-like tasks.

So the Astra solution is:

1. Keep Attribute Checks as the base engine.
2. Add a trained skill layer for repeatable competence.
3. Keep Techniques separate from skills.

## Distinction

- **Skills** are persistent trained competencies.
- **Techniques** are activated supernatural, martial, or metaphysical expressions.

That means:
- Stealth is a skill
- shadow-step is a Technique
- Medicine is a skill
- regrowing a limb is a Technique
- Survival is a skill
- beast-sense invocation is a Technique
