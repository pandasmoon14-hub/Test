# D01 Attribute Architecture, Visible Axes, Animus, Pneuma, and Breakthrough Lattices — Manifest

Version: v0.1  
Phase: Astra Ascension native design doctrine  
Status: Doctrine pack generated from accepted D01 decisions  
Primary owner: D01

## Purpose

D01 defines Astra’s character attribute architecture. It establishes the visible attributes, the Animus power/source axis, the hidden Pneuma world-facing pressure axis, the attribute breakthrough lattice model, and the handoff boundaries for derived pools and reserves.

D01 is not a complete character sheet, advancement system, resource economy, or transformation engine. It is the attribute doctrine that later files use.

## Attribute set

D01 accepts seven visible attributes and one hidden backend axis.

Visible attributes:

| Attribute | Function |
|---|---|
| Vessel | Body as container, force, capacity, form-change relevance. |
| Finesse | Speed, precision, kinetic control, fine action. |
| Reason | Cognitive processing, planning, interpretation, technical thought. |
| Insight | Perception, reading hidden state, intuition, assessment. |
| Integrity | Resilience, structural coherence, identity stability, endurance-like pressure. |
| Glamour | Social force, presence, illusion/performance/reality-adjacent impact. |
| Animus | Inner force, soul/spirit/will, cultivation power source axis. |

Hidden backend axis:

| Axis | Function |
|---|---|
| Pneuma | Fate/luck/cosmic breath pressure; not player-visible; feeds rare world-facing modifiers. |
