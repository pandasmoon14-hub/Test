# D01-01 Visible Attribute Set

## Purpose

This file defines the seven visible attributes.

## Vessel

Vessel is body as container, force, load-bearing, physical capacity, corporeal expression, and form-change relevance.

Vessel may inform:
- lifting, carrying, striking, grappling, physical exertion;
- bodily capacity;
- form-shift tolerance;
- body cultivation;
- corruption resistance by physical substrate when owner files validate it;
- platform/body interface where D08 applies.

Vessel is not donor Strength imported wholesale.

## Finesse

Finesse is speed, precision, reflex, kinetic control, dexterity, coordination, balance, and fine action.

Finesse may inform:
- precise movement;
- initiative/tempo pressure where D02/D07 later define it;
- stealthy or delicate methods through D05;
- ranged or controlled action;
- fine tool handling;
- vehicle/control interfaces where D05/D09 apply.

Finesse is not donor Dexterity imported wholesale.

## Reason

Reason is cognitive processing, planning, analysis, technical comprehension, memory organization, formal interpretation, strategy, and abstract problem solving.

Reason may inform:
- interpretation after Insight detects signal;
- research structure through D05;
- technical tasks;
- planning;
- focus pool handoffs if later authorized;
- inference from records and archives.

Reason does not replace Insight.

## Insight

Insight is perception, pattern recognition, intuition, hidden-state reading, emotional/environmental sensitivity, and assessment initiation.

Insight may inform:
- assessment action;
- reading opposition or place-state;
- detecting false information;
- sensing hidden pressure;
- noticing route/environment cues;
- social or spiritual perception where owner files validate it.

Insight probes. Reason interprets.

## Integrity

Integrity is resilience, structural coherence, identity stability, endurance-like pressure, continuity under strain, and resistance to breakdown.

Integrity may inform:
- vitality threshold;
- injury absorption;
- recovery pressure;
- corruption/backlash tolerance where D07 validates;
- form-state stability where D08 validates;
- identity stability;
- body/soul/structure coherence.

Integrity is not donor Constitution imported wholesale.

## Glamour

Glamour is social force, presence, projection, performance, charm, command aura, symbolic authority, mask, illusion-adjacent expression, and reality-facing presence.

Glamour may inform:
- persuasion and social pressure through D05;
- public identity and reputation pressure through D10;
- illusion/performance if routed to D06;
- faction presence;
- command reach;
- symbolic legitimacy.

Glamour is not donor Charisma imported wholesale.

## Animus

Animus is inner force, soul/spirit/will, cultivation pressure, personal power source axis, spiritual drive, and sustained self-projection.

Animus may inform:
- power pool handoffs;
- corruption resistance through D07 if validated;
- sustained focus;
- soul/spirit style action;
- route compatibility;
- breakthrough pressure;
- cultivation-style source models.

Animus is visible and player-facing.

## Attribute relation principles

1. Attributes overlap in application but remain distinct in doctrine.
2. Skill/method determines how an attribute is applied.
3. Route/Technique determines extraordinary expression.
4. Owner files validate derived pools, harm, advancement, actor changes, and world consequences.
5. No donor attribute label imports donor math.

## Acceptance criteria

The visible attribute set is valid when each attribute:

1. has a clear functional territory;
2. avoids donor ability-score leakage;
3. supports conversion from multiple genres;
4. can hand off to D03–D10 without owning their mechanics.
