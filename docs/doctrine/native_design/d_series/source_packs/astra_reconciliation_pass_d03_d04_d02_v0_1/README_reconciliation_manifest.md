# Astra Ascension — D03/D04/D02 Reconciliation Pass Manifest

Version: v0.1  
Status: amendment pack / pre-D06 gate  
Scope: D05-to-D03, D05-to-D04, and optional D02 relevance cross-check

## Purpose

This pack performs the required reconciliation after completion of D05: Competency, Profession, Knowledge, Training, Research, Synthesis, and Advancement Track doctrine.

It does not rewrite D03, D04, or D02 wholesale. It creates controlled amendment files that can be applied to the existing doctrine packs or used as patch instructions before beginning D06.

## Files

1. `D03_D05_reconciliation_amendment_v0_1.md`  
   Integrates D05 training, research, synthesis, method, competency, and advancement-track logic into D03 Power Economy and Expression doctrine.

2. `D04_D05_reconciliation_amendment_v0_1.md`  
   Integrates D05 competencies, professions, research, mastery, synthesis, and advancement-track routing into D04 proof, Development Picks, Awakening, tier benefits, breakthrough, and capstone doctrine.

3. `D02_D05_relevance_crosscheck_v0_1.md`  
   Adds an optional but recommended D02 cross-check for D05 relevance handling, lawful action vectors, check support, cost/consequence effects, and d20 roll-posture boundaries.

4. `reconciliation_manifest.json`  
   Machine-readable manifest for the reconciliation pass.

## Reconciliation principle

D05 is now the capability-substrate layer. It can record, support, route, and generate candidates. It does not authorize final Expression mechanics, Power Economy changes, breakthrough outcomes, Path/Class/Dao identity, item growth, companion growth, faction authority, territory state, or world-state changes.

D03, D04, and D02 must therefore be amended to reference D05 without surrendering their ownership.

## Required sequencing

Before D06 begins:

1. Apply or accept the D03 reconciliation amendment.
2. Apply or accept the D04 reconciliation amendment.
3. Review the optional D02 relevance cross-check.
4. Confirm that D06 may begin with a clean distinction between competency, profession, method, route pressure, Technique, Expression, and Path/Class/Dao.

## Non-ownership note

This reconciliation pack is not canon sourcebook prose, not final runtime schema, not live-play GM behavior, and not training/evaluation example material. It is design doctrine and patch guidance.
