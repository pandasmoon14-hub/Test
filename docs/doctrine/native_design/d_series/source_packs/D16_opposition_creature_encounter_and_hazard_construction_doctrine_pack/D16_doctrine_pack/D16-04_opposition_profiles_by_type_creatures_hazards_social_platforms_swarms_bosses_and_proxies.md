# D16-04 — Opposition Profiles by Type: Creatures, Hazards, Social, Platforms, Swarms, Bosses, and Proxies

Status: `doctrine-pack`
Version: `v0.1`
Layer: D16 / operational procedure doctrine
Primary scope: Opposition, creature, encounter, and hazard construction
Authority posture: Doctrine-facing and conversion-facing. Not final runtime schema, not final statblock schema, not encounter-balance math.

D16 is part of the D12–D18 operational procedure layer. It assumes D00–D15 are active doctrine dependencies and must preserve their owner boundaries.


## Purpose

Opposition Type Profiles are construction templates for recurring opposition families. They identify typical evidence, functions, pressure, handoffs, source-local risks, quarantine/escalation triggers, and rejected donor assumptions.

They are not final statblocks or subsystems.

## Core profiles

### Creature / Monster

Used for animals, monsters, beasts, spirits, aliens, undead-like entities, constructs, demons, aberrations, mutated beings, cultivation beasts, supernatural predators, and creature-like opposition.

Typical functions include harm, guard, pursue, ambush, control_area, create_exposure, inflict_condition, corrupt, test_intruders, protect_territory, and force_choice.

Primary handoffs: D08 actor substrate and creature ontology; D07 harm/conditions/corruption/exposure; D06 power expression if relevant; D10 source-local lore/ecology/territory/hidden truth; D12 cadence; D17 rewards/salvage only if relevant.

Rejected imports: donor creature type as Astra metaphysics, CR/level as Astra threat math, HP/armor/attack bonuses as Astra stats, immunity/resistance lists as final mechanics, monster ecology as canon without review.

### NPC / Rival

Used for named or unnamed individual opponents, rival Ascendants, agents, duelists, hunters, investigators, social rivals, bounty hunters, assassins, hostile mentors, and recurring personal opposition.

Handoffs: D08 actor/personhood; D05 method/competency; D06 Techniques/domains/oaths if relevant; D07 harm/condition; D10 relationship/faction/hidden truth/reputation; D15 institutional tie; D12 scene cadence.

Rejected imports: NPC class levels as Astra law, donor statblock as actor identity, social antagonism as combat reskin, loyalty or belief changes without state support.

### Hazard / Trap / Environmental Threat

Used for traps, terrain danger, environmental anomalies, storms, toxic zones, collapsing structures, unstable magic/technology, exposure fields, cursed terrain, security hazards, battlefield remnants, and site dangers.

Handoffs: D07 harm/exposure/conditions/corruption; D09 object/security/platform mechanisms; D10 site truth and hidden information; D12 hazard cadence; D14 exploration trigger/site-entry pressure.

Rejected imports: trap DCs as Astra DN law, trap damage dice as Astra harm law, room-entry trigger as automatic reveal, hazard statblock as universal schema.

### Social Antagonist / Legal Blocker

Used for judges, officials, hostile patrons, guild clerks, public accusers, faction envoys, gatekeepers, propagandists, investigators, social rivals, blackmailers, and legal/institutional obstacles.

Handoffs: D10 law/public belief/rumor/relationship/hidden truth; D11 presentation/misinformation boundaries; D12 debate/confrontation/trial/negotiation cadence; D15 institutional operation; D17 licenses/bribes/payments/requisition if relevant.

Rejected imports: social HP as default law, single persuasion check as automatic success, relationship tags as mind control, public belief as truth, donor legal code as Astra law.

### Security / Object-System

Used for locks, alarms, sensors, drones-as-system, automated defenses, wards, surveillance webs, containment fields, firewall-like systems, doors, vaults, object-tied traps, and platform subsystems.

Handoffs: D09 object/tool/platform/security state; D05 method/technical procedure; D10 security ownership and hidden information; D12 active scene transition; D13 repair/bypass project if interval-scale; D17 acquisition/licenses/requisition if access or equipment matters.

Rejected imports: donor hacking rounds as Astra law, lock DCs as universal access math, security rating as Astra schema, object system as creature statblock.

### Swarm / Horde / Group

Used for mobs, crowds, insect swarms, zombie hordes, drone clouds, squads, militia groups, cultist masses, wild packs, and social crowds.

Handoffs: D08 whether group is actor collective or many actors; D07 harm/exposure/condition routing; D10 crowd/faction/public-state implications; D12 cadence and multi-actor sequencing; D18 long-horizon mass pressure if recurring.

Rejected imports: swarm damage math as universal rule, minion rules as default law, mob morale as automatic rule, crowds as single NPCs without boundary.

### Elite / Boss / Dominant Threat

Used for encounter-defining opposition, major rivals, apex creatures, dominant hazards, named bosses, high-tier guardians, faction champions, platform command threats, and source-local set pieces.

Handoffs: D06 power/domain/Technique expression; D07 harm/backlash/corruption/condition consequences; D08 actor/substrate/personhood; D09 object/platform ties; D10 lore/hidden weakness/faction/world-state; D12 cadence/reactions/interruptions; D18 recurring threat/arc pressure if campaign-defining.

Rejected imports: legendary actions as Astra default, boss phases as universal structure, solo math as encounter balance, lair actions as default law, scripted boss immunity as unsupported mechanics.

### Vehicle / Ship / Mech / Platform Enemy

Used for vehicles, starships, mechs, mobile fortresses, stations, bases, turrets, drones as platforms, war machines, domain infrastructure, and large support assets.

Handoffs: D09 platform/object state/modules/hardpoints/systems; D08 crew/actor relations; D03 fuel/power/heat/strain if supported; D07 platform harm and actor harm consequences; D12 platform/crew cadence; D14 travel/vector/site pressure; D17 fuel/maintenance/salvage/requisition.

Rejected imports: ship turns as Astra action economy, hull points as default HP, hardpoints as universal platform schema, vehicle speed/range math as Astra law, platform enemy as giant creature.

### Summon / Companion Under Opposition Role

Used when a companion, summon, familiar, controlled entity, drone, spirit, clone, pet, minion, or allied asset acts as opposition due to control, corruption, command conflict, hostile ownership, or source-local procedure.

Handoffs: D08 companion/summon/personhood/substrate/control boundaries; D06 summoning/control/power expression; D07 harm/condition/corruption; D10 relationship/world-state; D12 encounter cadence.

Rejected imports: summons as disposable statblocks by default, companion loyalty as automatic, control rules as donor metaphysics, pet/familiar schema as universal law.

### Institutional Proxy / Faction Agent

Used for patrols, officers, guild agents, corporate teams, sect enforcers, legal representatives, military detachments, informants, bounty teams, domain guardians, and other institutional pressure made encounter-facing.

Handoffs: D10 faction/law/world-state; D15 institutional operation and pressure; D08 individual agents as actors; D12 scene cadence; D17 bribes/licenses/requisition if relevant; D18 campaign-scale institutional conflict if recurring.

Rejected imports: faction agents as random combat encounters by default, law enforcement as universal moral authority, patrol tables as default procedure, institutional response as omniscient.
