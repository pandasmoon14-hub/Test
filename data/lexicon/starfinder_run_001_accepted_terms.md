# Starfinder Run 001 Accepted Terms

Run ID: `starfinder_prod_run_001`  
Purpose: `Accepted Astra-facing lexicon ledger derived from the Starfinder run after merge review and integration.`

This file records terms that are safe to carry forward from the Starfinder run into Astra-facing review work.
It does **not** promote exact setting cosmology, exact ship math, or exact donor catalogs.
