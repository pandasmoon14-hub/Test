# RHBF Run 002 Accepted Terms

Run ID: `rhbf_prod_run_002`  
Donor: `Ruthless Heavens, Boundless Fate`  
Purpose: `Record the RHBF-derived terms accepted into Astra-facing lexicon, along with their promotion status and notes.`

---

## 1. Use rule

Only include terms here if they were accepted through merge review as either:

- **Astra baseline grammar**, or
- **Astra optional profile**.

Do **not** place donor-local payload terms here merely because they appeared in RHBF.

## 4. Terms deliberately excluded from accepted lexicon

The following are intentionally **not** promoted into the accepted lexicon from RHBF at this stage:

- exact RHBF faction names,
- exact RHBF proper nouns tied to donor politics,
- exact anti-technology ideology terms,
- exact `1/4/1/3` technique-slot profile wording as Astra law,
- exact reserve-triplet payload terms as settled Astra resource law,
- exact Bound Relic math vocabulary tied to RHBF caps,
- exact wondrous-item catalog names,
- exact donor component names where broader grammar already exists.

---

## 5. Lexicon handling note for later donors

Later donors may:

- reuse the accepted baseline terms freely where the construct is genuinely the same,
- reuse optional-profile terms only where the donor pattern truly matches,
- propose alternate terms where RHBF only validated the grammar but not the donor label,
- challenge whether some optional-profile terms deserve future promotion to baseline.
